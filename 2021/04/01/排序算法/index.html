排序算法

这里说的数字，代表一个对象的特征码

先来第一个

直接选择插入

这类排序的方法比较en…..当时我是理解错了，导致写不出来。原理就是先从第一个数字开始，前i个数字已经排序好了（第一个i=0）所以第i+1个数向前和前i个数比较，比较的时候把自己的位置当成空位，如果比自己大就和它换个位置，知道遇到比自己小的数字，就停下，插入到当前的空位。

算法实现（C++）

template <class T>
void insert_sort(T A[],int n)
{
    for(int i = 0; i<n; i++0)
    {
        T tmp = A[i];
        j = i-1;
        while(j>=0 && A[i]>A[j])
        {
            A[j+1] = A[j];//本质是相邻位置的移动，i不能改变
            j--;
        }
        A[j+1] = tmp;
    }
}

这个算法其实很简单，重点还是在于，抓住相邻位置交换的本质，和从后面扫描的简化，而不是从前面扫描。

折半选择插入

原理和上面一样，区别在于，折半这两个字，每次都是在前面的i个数中折半去查找合适的插入位置。

template <class T>
void BinaryInsertionsort(T A[], int n)
{
    for(int i = 0; i<n; i++)
    {
        T tmp = A[i];
        int j = i-1;
        top = j;
        low = 0;
        while(top>=low)
        {
            int mid = (top+low)/2;
            if(A[mid] > A[i])
            {
                top = mid-1;
            }
            else {low = mid+1;}
        }
        //插入位置为low
        for(int m = i; m>=low; m--)
        {
            A[m] = A[m-1];
        }
        A[m] = tmp;
    }
}

关键点在于这里终止条件的判断，和mid不符合之后上下取数的变化，记住最好。

shell排序

这个排序的关键点在于，分组，最后一次所有的再次排序保证了程序的正确性，而前面的分组，是在减少最后一次的交换操作的次数。

template <class T>
void shell_sort(T A[], int n,int s)//s为初始增量，每循环一次减半
{
    int i,j,k;
    T tmp;
    for(k = s;k>0;k>>=1)//k为遍历量
    {
        //分组排序，以k为一组，从第k个开始，和i-k比较，保证了每次都是组内排序，并且是k组交替排序。
        //这一步处理的很好
        for(i=k;i<n;i++)
        {
            tmp = A[i];
            j = i-k;
            //组内排序，将temp直接插入组内合适的位置记录
            while(j>=0 && temp<A[j])
            {
                A[j+k] = A[j];
                j-=k;
            }
            A[j+k] = tmp;
        }
    }
}

快速排序

这种排序利用的是一种，分治的思想，先去随便选择一个数，然后把比他小的放在它前面，比它大的放在它后面，之后再分别对前后两个部分做相同的操作，知道前后分组为0，也就是完全排好序。

所以这个算法的核心就在于，如何把比这个数大和小的那些数字分别放到这个数的两侧。

这里又来了一种算法，以前没见过，也算是一种学到的比较方法了。

  • 首先，随便找一个数，（找第一个）[狗头]
  • 然后，这个数字相当于空出来了，然后从最后一个去和这个数比较，直到找到比这个数小的数，然后交换
  • 之后空位相当于发生了改变，然后再从前面开始搜寻比它大的数字。操作同上
  • 之后就是循环操作
  • 然后就是分组操作。

下面是算法实现。

//快速排序
void quick_sort(int s[], int l, int r)
{
    if (l < r)
    {
        //Swap(s[l], s[(l + r) / 2]); //将中间的这个数和第一个数交换 参见注1
        int i = l, j = r, x = s[l];
        while (i < j)
        {
            while(i < j && s[j] >= x) // 从右向左找第一个小于x的数
                j--;  
            if(i < j) 
                s[i++] = s[j];
            
            while(i < j && s[i] < x) // 从左向右找第一个大于等于x的数
                i++;  
            if(i < j) 
                s[j--] = s[i];
        }
        s[i] = x;
        quick_sort(s, l, i - 1); // 递归调用 
        quick_sort(s, i + 1, r);
    }
}

交换排序（冒泡排序）

这个比较简单了，从第一个开始逐次和它后面的一个数字比较，小于自己则交换位置，那么第一次交换之后，最后的就是最大值。所以循环比较，每次都获得最大的数字。

template <class T>
void BubbleSort(T A[], int n)
{
    int i,j;
    bool flag;
    T temp;
    for(int i = n-1, flag=true; i>0&&flag; i--)
    {
        //设置未交换标志
        flag = false;
        for(j = 0; j<i; j++)
        {
            if(A[j+1]<A[j])
            {
                flag = false;
                temp = A[j+1];
                A[j+1] = A[j];
                A[j]=temp;
            }
        }
    }
}
